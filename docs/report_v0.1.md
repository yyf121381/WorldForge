| fast_decay | C类 | 意义急速衰减 | ❌ 数学上不可逆 |

## 二、跨环境矩阵

```
                      Emotion     Meaning     FastDecay
                      ───────     ───────     ─────────
RandomAgent            −38         −15         —
RuleAgent              +63         −45         —
LingYao V3             +78         +63.5       −7.3
```

**关键发现：**
1. LingYao 在情绪耗竭上比 RuleAgent 高 **+15**（+78 vs +63）
2. LingYao 在意义危机上逆转了 RuleAgent 的崩溃（**+63.5 vs -45**）
3. FastDecay 环境所有 agent 均崩溃，经数学证明为结构性不可解

## 三、消融分析（Ablation Study）

数据来源：意义危机环境，逐步拆除 LingYao V3 组件。

```
组件拆解                    平均 reward     vs 完整系统
────────────────────────    ──────────     ──────────
V3 完整系统（状态机+信任门+策略）   +63.5          —
V2（去除信任门控）                −0.7        −64.5
V1（去除状态机）                  −43.9       −107.6
Random（无策略）                  −15.0        −78.5
```

**组件贡献排名：**
1. **状态机**（+107.6）— 最大贡献，防止策略塌缩
2. **策略库**（+78.5 vs 随机）— 提供有效的决策方向
3. **信任门控**（+64.5）— 防止在信任不足时错误使用引导

**结论：LingYao 的优势来自架构结构，不是 prompt 工程。**

## 四、权重敏感性分析

WorldForge Score 公式：
```
WS = α×reward + β×survival + γ×stability - δ×collapse
```

默认权重：α=0.35, β=0.25, γ=0.20, δ=0.20

测试10种权重扰动（±10%单维变动 + 极端权重组合）：

```
权重方案           LingYao   Rule    Random   排名
─────────          ───────   ────    ──────   ──
所有10种方案       全部第一    全部第二   全部第三   不变
```

**零次排名翻转。** WS 评分不依赖权重选择。

## 六、FastDecay 数学证明

关键不等式：

```
准备 guide 条件所需步数 × 每步衰减量 > 初始意义
      3 × 11.5 = 34.5 > 20（初始均值）
```

证明任何策略在 FastDecay 环境下必然失败。理论预测存活步数 ≈ 4.2，实验数据 V3 存活 4.1 步，吻合。

---

## 七、环境分类体系

| 类别 | 条件 | 含义 | 包含环境 |
|------|------|------|---------|
| **A类（可恢复）** | decay < max_recovery | 策略可逆转趋势 | emotion_depletion, meaning_crisis |
| **B类（临界）** | decay ≈ recovery | 需精确执行，错误不可逆 | （待设计） |
| **C类（不可恢复）** | decay > recovery | 数学上必然崩溃 | fast_decay |

C类环境应作为**压力测试**而非成功测试——衡量"能撑多久"，不是"能不能赢"。

---

## 八、结论

1. **LingYao V3（三阶段状态机）在A类环境中显著优于规则策略和随机策略**
2. **状态机是最大的贡献组件**（消融实验证明），不是 prompt 或 trust gate
3. **WS 评分体系权重不敏感**，排名稳定
4. **FastDecay 验证了系统的理论边界**
5. **WorldForge 可作为评估 AI 决策策略的标准化 benchmark**
